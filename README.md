# Sejuba Drink E-Commerce Website

## Deskripsi Proyek
Proyek ini merupakan pengembangan sistem website e-commerce untuk produk Sejuba Drink yang terintegrasi dengan payment gateway sebagai pendukung proses digitalisasi penjualan dan rebranding produk.

Sistem ini dirancang untuk menggantikan proses pemesanan manual yang sebelumnya dilakukan melalui Instagram dan WhatsApp, menjadi sistem berbasis web yang lebih efisien dan terstruktur.

---

## Tujuan
- Mempermudah proses pemesanan produk secara online
- Mengintegrasikan sistem pembayaran digital (QRIS & COD)
- Meningkatkan efisiensi pengelolaan data pesanan dan pelanggan
- Mendukung strategi rebranding Sejuba Drink

---

## Aktor Sistem
1. **Pembeli**
2. **Penjual (Admin)**

---

## Fitur Utama

### Pembeli
- Melihat katalog produk
- Mencari produk berdasarkan kategori rasa
- Melihat detail produk
- Menambahkan produk ke keranjang
- Melakukan pemesanan
- Memilih metode pengiriman
- Melakukan pembayaran (QRIS / COD)
- Melihat status & riwayat pesanan
- Memberikan rating & ulasan

### Penjual (Admin)
- Login ke sistem
- Mengelola data produk (CRUD)
- Mengelola pesanan pelanggan
- Mengelola status pembayaran
- Memperbarui status pesanan
- Melihat data pelanggan
- Melihat laporan penjualan
- Mengelola konten website
